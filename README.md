# タスク管理アプリ

##  機能説明
googleアカウントでログイン、ログアウト機能付き
テキストと日付を入力送信でタスクを管理できる
タスクは納期日と残り日数を表示できる
タスク完了未完了をでチェックで管理できる
タスク並び替え機能付き（追加順、納期順、未完了優先）
タスクを編集、削除できる




##  使用技術スタック 
* フロントエンド React (TypeScript) + Vite
* スタイリング Tailwind CSS + DaisyUI
* バックエンド Node.js + Express
* データベース　MySql
* Google連携　firebase
* 環境変数管理 dotenv


##  実行環境
MacBook Air　MacOSバージョン14.5 メモリ8G
